# WeatherForecast

A weather forecasting application using OpenWeatherMap API, ImGui, and C++.

## Dependencies

- CMake
- OpenGL
- GLFW
- GLEW
- ImGui
- nlohmann::json
- httplib

## Building

```bash
git clone https://github.com/yourusername/WeatherForecast.git
cd WeatherForecast
mkdir build
cd build
cmake ..
make
